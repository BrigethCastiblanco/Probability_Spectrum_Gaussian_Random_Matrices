# Probability_Spectrum_Gaussian_Random_Matrices
Este repositorio contiene los códigos utilizados en el documento del proyecto final de grado con los que se simularon los espectros de probabilidad de los tres ensambles clásicos (GOE, GUE, GSE) para llegar a la ley del semicírculo de Wigner. Sin embargo, los datos reportados en el documento no se pudieron subir debido a su tamaño, pero en las carpetas data y gráficas se incluyen algunos datos de ejemplo para el ensamble GUE que reflejan la evolución del ensamble hacia el semicírculo a medida que aumenta la dimensión de sus matrices.
